## Como instalar os Arquivos

- `npm init -y` - _Instala o package.json_

- `npm i sequelize express mysql` - _Dependencias usadas.Use o banco que desejar, eu usei o **mysql**_

- `npm i nodemon -D` - _DevDependencia usada_
